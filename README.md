📊 iPhone 17 & 17 Pro Sales Analysis Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard analyzing the sales performance of iPhone 17 and iPhone 17 Pro.
The dashboard transforms raw sales data into actionable insights by analyzing revenue trends, units sold, regional performance, storage preferences, and marketing efficiency.

The goal of this project is to demonstrate data analysis, data visualization, and business insight generation using Power BI.

🎯 Project Objectives

Analyze the overall sales performance of iPhone 17 and iPhone 17 Pro

Compare units sold and revenue contribution by model

Identify regional sales distribution

Evaluate storage variant demand

Analyze revenue trends over time

Measure marketing efficiency using ROAS (Return on Ad Spend)

📊 Dashboard KPIs

The dashboard includes the following key metrics:

Total Revenue

Total Units Sold

Revenue per Unit

Return on Ad Spend (ROAS)

These KPIs provide a quick overview of overall business performance.

📈 Key Dashboard Insights
📱 Model Performance

Comparison of iPhone 17 vs iPhone 17 Pro based on units sold and revenue.

🌍 Regional Analysis

Identifies top-performing regions contributing to overall sales.

💾 Storage Variant Analysis

Analyzes customer preference across storage options:

128GB

256GB

512GB

1TB

📅 Revenue Trend

Tracks revenue changes over time to identify sales patterns and fluctuations.

📊 Market Share

Visualizes the distribution of sales between the two models.
